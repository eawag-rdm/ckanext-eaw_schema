Status

The status of the package

incomplete
Either the package metadata or the actual data (the ressources) or both will be amended in the future.
complete
The package is finished and no changes are expected anymore.
reviewed
The package has been additionally reviewed for completeness and comprehensibility by someone who is familiar with the field but not identical to the submitter.
