{# Help text for variables field for package create/update #}

Variables

Select the variable(s) for which measurements are contained in the package.

This is a controlled vocabulary and you can only select pre-defined terms.

It is likely that variables you measured are not yet in this list. In this case, please note that in the Notes-field.

As a rule of thumb, everything that has a unit is potentially a Variable.

In case you can think of relevant terms for this field that should be added to the list but do not pertain to this package specifically, please note them enclosed in parentheses: "(New_Variable)".

This is a mandatory field.
Select none if and only if Variables does not apply to this package.
