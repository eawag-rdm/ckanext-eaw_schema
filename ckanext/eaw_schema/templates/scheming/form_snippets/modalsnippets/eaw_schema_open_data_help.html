Open Data

Indicates that this data package should be published available for anyone (also outside Eawag). This should be the norm.

If this is set to "YES" and the package is complete, please

  1. review the package (quality, completeness, legal aspects)
  2. download the Checklist for publishing data packages at Eawag (PDF) and tick the appropriate boxes.
  3. send the form to rdm@eawag.ch

We will then

  • assign a Digital Object Identifier (DOI),
  • publish it and
  • email you the DOI-URL that points to the package.

If you need a DOI to put into the manuscript (e.g. under Data Availability) before the data package is ready for publication, please get in touch with rdm@eawag.ch. We will then reserve one for you, but it will only be activated once the complete package has been published.
If you need this package to be accessible from outside Eawag now, before it is ready for final publication, for example if that is a requirement for review of the associated paper, please send an email to rdm@eawag.ch
