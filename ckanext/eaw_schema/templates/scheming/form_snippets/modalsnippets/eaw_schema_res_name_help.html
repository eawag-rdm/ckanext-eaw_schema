Resource Name

A descriptive name for the resource.

May be composed of alphanumeric characters and the special characters "$-_.+!,".

Usually the filename is a reasonable choice. It can be sensible to shorten very long filenames to use them as Name. The purpose of the Name is to give the user quickly a coarse idea about the content of the resource. It can be helpful to keep the extension of the filename to indicate the file format, e.g. *.pdf, *.R, *.csv, *.svg, *.png ...

This is a mandatory field.
