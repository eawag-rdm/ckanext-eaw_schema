Review Level

Data packages need to be reviewed to ensure an acceptable level of quality. Of course, a review can not be conducted by the person uploading the dataset.
none
The package was not yet reviewed.

This should only be a temporary state of affairs.

general
The package has passed a general consistency- and completeness check, as described in the Checklist for publishing data packages at Eawag .
All data packages should be reviewed as soon as possible after the upload.
domain specific
The package was checked by a domain expert for re-usability in a scientific context.
Data packages to be published as Open Data should be reviewed by a domain expert.
