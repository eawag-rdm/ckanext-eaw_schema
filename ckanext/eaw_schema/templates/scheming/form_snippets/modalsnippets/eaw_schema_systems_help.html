{# Help text for variables field for package create/update #}

Systems

Select the system(s) to which the data in this package refers to.

This is a controlled vocabulary and you can only select pre-defined terms.

It is likely that one or several Systems, which would make sense for this package, are not yet in this list. In this case, please note that in the Notes-field.

In case you can think of Systems that should be added to the list but do not pertain to this package specifically, please note them enclosed in parentheses:
"(New_System)".

As a rule of thumb, System denotes a spacial and/or functional delineation of the realm in which measurements took place. Prototypical Systems in that sense are the environmental compartments.

This is a mandatory field.
Select none if and only if Systems does not apply to this package.
