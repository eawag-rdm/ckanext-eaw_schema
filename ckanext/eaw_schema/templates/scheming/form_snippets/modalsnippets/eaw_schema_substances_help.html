Substances

List the chemical substances (molecules, ions, elements) that is(are) the subject of this package.

Substances should be denoted by a commonly understood (by chemists) name and be as unambiguous as possible. If in doubt, please refer to the IUPAC recommendations, which can be found for example in the PubChem Compound Database.

You can write (or copy & paste) a list of substances into the text field, which expects one item per line.
