Title & URL

A well chosen title for this data package.

In case you are creating a publication data package, i.e., a package that supplies data for a specific publication, the title should be of the form:

"Data for: <title of associated publication>"

The URL is being filled in automatically, but you can change it to any not yet used value that makes sense as URL. That is useful if you have a rather long title and get an error message that complains about a too long URL: You might just want to truncate it.

This is a mandatory field.
