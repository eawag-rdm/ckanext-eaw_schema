Resource Type

The type of the resource is a general description of the file's content.

This is a controlled vocabulary and you can only select pre-defined terms.

A zip-archive containing files of different resource types is technically a Collection. However, if the main content can be characterized with one Resource Type, choose that one.

E.g., assign Resource Type = "Dataset" to a zip-archive that contains a csv-file with numerical content and a README.txt and an ancillary photo of the measurement site.

This is a mandatory field.
