Usage Contact

The person who has to be contacted before the data in this package can be used.

This is usually the first author of the package. However, if it is foreseeable that this researcher will leave Eawag in the near future (e.g. a PhD student), please enter the respective group leader.

Firstname1(s) Lastname <email@domainname.tld>
